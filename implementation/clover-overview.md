# Clover Integration — Overview

## Purpose
This document provides a high-level overview of the Clover POS integration with the
Proscai ERP platform.

> **Note:** This is a sanitized portfolio example. System names and workflows are
generalized to preserve confidentiality.

---

## What Is Clover?
Clover is a point-of-sale platform that combines hardware (terminals, scanners,
payment devices) with cloud-based services for payments and order management.

---

## Integration Goals

The integration enables:
- Real-time item scanning visibility
- Customer-facing order display
- Loyalty program interaction
- Secure payment method selection
- Reduced cashier friction during checkout
